## Features
- Fully generated C# SDK based on the [D-ID API](https://docs.d-id.com/reference/get-started) using [AutoSDK](https://github.com/HavenDV/AutoSDK)
- Talking avatar videos, AI agents, streaming conversations, and animations
- Same day update to support new features
- Updated and supported automatically if there are no breaking changes
- All modern .NET features - nullability, trimming, NativeAOT, etc.

### Usage
```csharp
using DId;

using var client = new DIdClient(apiKey);
```
